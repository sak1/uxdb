# UXDB实例

UXDB(uxsinoDB)是国内新生的商业数据库，为增进大家对这款数据库的了解，我们对数据库的安装部署、库表操作、SQL编程、调优、备份恢复等进行了实例验证，结合案例对不同业务场景下应用进行了说明。希望为程序员和DBA朋友们提供些帮助，全文100余例。Maverick、大雄对此亦有贡献，诚谢  ——Steven 2020-2021
